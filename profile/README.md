# Cast CLI — The DevSecOps Governance Toolkit

**AI can generate a pipeline that runs. Cast CLI enforces a pipeline that complies.**

[![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)](LICENSE)
[![PyPI version](https://img.shields.io/pypi/v/castops.svg)](https://pypi.org/project/castops/)

</div>

Cast CLI enforces pipeline standards across GitHub Actions & GitLab CI without manual reviews. Ship code securely and uniformly — with built-in audit, upgrade, and org-level governance.

DevOps/Platform engineer best practices, translated into executable CI/CD baselines: one command to install, continuous audit to enforce, and automated upgrade to stay current.
